Installation:
=============
Simply send the including .8xk to your TI-83+/SE or TI-84+/SE graphing calculator.
Run Doors CS via [Apps] --> DoorsCS7. Full documentation at
http://dcs.cemetech.net; discussions at http://www.cemetech.net/forum. Only use
the contents of the Other Versions/TI-83/ folder if you have a TI-83 (NOT a
TI-83+, TI-83+ SE, TI-84+, or TI-84+SE). For a TI-84 Plus C Silver Edition
(color-screen) version of Doors CS, visit http://www.cemetech.net or
http://dcs.cemetech.net and look for news.

Read the DCS7_Manual.pdf file included for full installation guide, usage
tutorial, description of shortcut keys, and more.

Known Issues
============
No significant bugs are known at this time.  It is known that even without
Doors CS, TI-OS 2.53MP will corrupt BASIC programs in certain rare low-memory
situations.  It is recommended that backups be used, and that due to widespread
bugs in the OS, that 2.53MP be avoided.
